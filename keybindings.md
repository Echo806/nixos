# Niri 快捷键

Mod = Super/Windows

## Noctalia IPC

| 快捷键 | 功能 |
|--------|------|
| `Mod+Space` | 启动器 |
| `Mod+Escape` | 会话菜单（锁屏/注销/关机） |
| `Mod+L` | 锁屏 |
| `XF86AudioLowerVolume` | 音量减 |
| `XF86AudioRaiseVolume` | 音量增 |
| `XF86AudioMute` | 静音 |
| `XF86MonBrightnessDown` | 亮度减 |
| `XF86MonBrightnessUp` | 亮度增 |

## 启动应用

| 快捷键 | 应用 |
|--------|------|
| `Mod+T` | Ghostty 终端 |
| `Mod+B` | Chrome 浏览器 |

## 窗口管理

| 快捷键 | 功能 |
|--------|------|
| `Mod+Q` | 关闭窗口 |
| `Mod+Shift+E` | 退出 niri |
| `Mod+F` | 全屏切换 |

## 焦点切换

| 快捷键 | 功能 |
|--------|------|
| `Mod+←` | 聚焦左侧列 |
| `Mod+↓` | 聚焦下方窗口 |
| `Mod+↑` | 聚焦上方窗口 |
| `Mod+→` | 聚焦右侧列 |
| `Mod+Home` | 聚焦首列 |
| `Mod+End` | 聚焦末列 |

## 移动窗口

| 快捷键 | 功能 |
|--------|------|
| `Mod+Ctrl+←` | 移动到左侧列 |
| `Mod+Ctrl+↓` | 向下移动窗口 |
| `Mod+Ctrl+↑` | 向上移动窗口 |
| `Mod+Ctrl+→` | 移动到右侧列 |

## 工作区

| 快捷键 | 功能 |
|--------|------|
| `Mod+U` | 切换到上方工作区 |
| `Mod+D` | 切换到下方工作区 |
| `Mod+Ctrl+U` | 移动窗口到上方工作区 |
| `Mod+Ctrl+D` | 移动窗口到下方工作区 |

## 截图

| 快捷键 | 功能 |
|--------|------|
| `Print` | 截图 |

## 布局

| 快捷键 | 功能 |
|--------|------|
| `Mod+.` | 吸收窗口到当前列 |
| `Mod+,` | 从列中弹出窗口 |
| `Mod+Shift+←` | 聚焦左侧显示器 |
| `Mod+Shift+→` | 聚焦右侧显示器 |
| `Mod+Shift+↑` | 聚焦上方显示器 |
| `Mod+Shift+↓` | 聚焦下方显示器 |
| `Mod+Ctrl+Shift+←` | 移动列到左侧显示器 |
| `Mod+Ctrl+Shift+→` | 移动列到右侧显示器 |
| `Mod+Ctrl+Shift+↑` | 移动列到上方显示器 |
| `Mod+Ctrl+Shift+↓` | 移动列到下方显示器 |
| `Mod+-` | 减小列宽 5% |
| `Mod++` | 增加列宽 5% |
| `Mod+Shift+-` | 减小窗口高度 5% |
| `Mod+Shift++` | 增加窗口高度 5% |
